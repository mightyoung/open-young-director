## S01 矿山宿舍（黎明）

### 场景基础信息
- **ID**: S01
- **名称**: 矿山宿舍
- **时段**: 黎明（天色微明）
- **光线**: 低光照，昏黄油灯光

### 场景Prompt（内景）

```
【技术参数】9:16竖屏，8K超清，电影级画质
【风格】高度写实风格
【绝对主体】简陋矿工宿舍内部，昏暗压抑
【空间布局】狭小空间，约10平米；土坯墙，墙面斑驳脱落；屋顶为木质横梁+茅草，有裂缝透光
【道具陈设】
  - 土炕（北方硬床），铺旧草席，被褥单薄破旧
  - 木凳1个，缺腿用砖垫起
  - 角落木箱（存放衣物）
  - 墙上挂油灯，火焰摇曳，投影晃动
  - 地面为夯土，有裂缝
【环境细节】
  - 墙角有蜘蛛网
  - 墙面有汗渍水痕
  - 窗户为木框糊纸，透出微弱灰白天光
【光线设计】
  - 主光源：油灯（暖黄色，2700K），照亮中心区域
  - 辅光源：窗户透光（冷蓝色，6000K），照亮墙面裂缝
  - 光影：油灯造成强烈明暗对比，阴影区域约占40%
【氛围】阴暗、潮湿、压抑、贫困
【时间线索】黎明，窗外天色微明但室内仍昏暗
【负面提示词】bright, clean, modern, luxury, anime, cartoon, CGI
```

## S02 临渊镇街道（清晨）

### 场景基础信息
- **ID**: S02
- **名称**: 临渊镇街道
- **时段**: 清晨（太阳初升）
- **光线**: 晨曦微光，薄雾

### 场景Prompt（外景）

```
【技术参数】9:16竖屏，8K超清，电影级画质
【风格】高度写实风格，仙侠古风
【绝对主体】偏僻山乡小镇清晨街道，晨雾缭绕
【空间布局】
  - 街道为青石板路，宽约3米，有积雪/泥泞痕迹
  - 两旁为低矮民居，土坯墙，黑瓦屋顶
  - 远处可见连绵天荡山脉轮廓
  - 街道尽头隐约可见矿山入口
【建筑细节】
  - 民居门扉陈旧，贴有褪色春联
  - 墙角堆放柴火
  - 偶有鸡犬走动
  - 远处烟囱冒出炊烟
【环境细节】
  - 薄雾笼罩，能见度约50米
  - 地面微湿（有露水）
  - 空气清冷，可见呼吸白雾
【光线设计】
  - 主光源：晨曦天光（冷蓝色，5000K），从山脉方向照射
  - 辅光源：民居窗户透出暖黄灯光（2700K）
  - 光线穿透薄雾，形成体积光效果
【氛围】宁静、古朴、清冷、略带神秘
【时间线索】清晨太阳初升，雾气未散
【负面提示词】modern, bright sunshine, clean streets, anime, cartoon, CGI
```

## S03 矿山入口（清晨）

### 场景基础信息
- **ID**: S03
- **名称**: 矿山入口
- **时段**: 清晨（太阳升起）
- **光线**: 晨光，雾气较重

### 场景Prompt（外景）

```
【技术参数】9:16竖屏，8K超清，电影级画质
【风格】高度写实风格，仙侠古风
【绝对主体】矿山入口广场，人群聚集
【空间布局】
  - 入口为半圆形矿洞洞口，宽约5米，高约4米
  - 洞口用木支架支撑，显示年代久远
  - 洞口上方岩石刻有"临渊矿"三字（模糊斑驳）
  - 广场为碎石地面，面积约200平米
  - 三面环山，晨雾缭绕
【设施道具】
  - 入口两侧竖立火把架（白天熄灭）
  - 广场边缘有登记棚屋（茅草顶，木桌）
  - 管事站立处有简单遮阳棚
  - 远处有矿车轨道延伸入洞
【人群分布】
  - 童工（约20-30人）排成长队
  - 管事（3-4人）穿黑色制服巡视
  - 护卫（2人）持短刀站立
【光线设计】
  - 主光源：晨曦天光（冷蓝色，5000K），从洞口内侧照射
  - 辅光源：前方山体反射光
  - 光影：洞口内暗外亮，形成强烈明暗对比（剪影效果）
【氛围】压抑、紧张、等级分明
【时间线索】清晨，雾气从洞口向外飘散
【负面提示词】modern, bright, clean, anime, cartoon, CGI
```

## 场景使用说明

### 在镜头Prompt中引用场景
```markdown
【素材引用】
@图片01[S01]  矿山宿舍全景（黎明氛围）
@图片01[S01-D]  矿山宿舍细节（油灯/草鞋）
@图片02[S02]  临渊镇街道全景（晨雾）
@图片02[S02-F]  临渊镇街道氛围（晨雾特写）
@图片03[S03]  矿山入口全景
@图片03[S03-M]  矿山入口人群（童工/管事）
```

### 场景连续性规则
1. **同一场景必须保持光线/时间一致性**
2. **场景引用ID在同场景的所有镜头中保持一致**
3. **场景细节变体用后缀区分**（-D=细节，-F=氛围，M=人群）
