# 《穿越古代修仙》衔接图Prompt

> 用于视频生成时的首帧/尾帧衔接图
> 格式：4K高清衔接图，用于即梦【图生图】→【图生视频】流程

---

## 衔接图使用说明

### 工作流
```
镜头N尾帧描述 → 撰写衔接图Prompt → 即梦【文生图】生成4K衔接图 → 【图生视频】生成镜头N+1
```

### 质量标准
| 项目 | 要求 |
|------|------|
| 分辨率 | 4K (2160p) |
| 风格 | 高度写实 |
| 光线 | 与尾帧一致 |
| 姿势 | 延续尾帧最后一帧 |
| 表情 | 自然过渡 |
