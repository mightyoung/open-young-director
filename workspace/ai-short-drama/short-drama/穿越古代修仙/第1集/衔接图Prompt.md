## 衔接图使用流程

```
1. 镜头N尾帧 → 提取尾帧描述
2. 根据尾帧描述撰写衔接图Prompt
3. 即梦【文生图】生成4K衔接图
4. 衔接图 + @图片[C01] → 即梦【图生视频】生成镜头N+1
```
