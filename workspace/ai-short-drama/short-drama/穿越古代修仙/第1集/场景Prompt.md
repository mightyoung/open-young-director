```

简陋破旧的木结构房屋
斑驳的石墙，有水渍和裂缝
潮湿阴冷，空气中弥漫着尘土味
一张破旧木床，草垫铺就
墙角堆放劳作工具

幽暗压抑氛围
负面：动漫，干净明亮，现代设施
```

**床铺细节**
```
写实摄影，超高清，破旧木床特写
粗糙木架结构，草垫床铺
破旧被褥，颜色发灰
床头有小木桌，放置油灯
远处石墙斑驳
阴暗氛围

## S02 临渊镇街道（清晨）

| 属性 | 内容 |
|------|------|
| 场景类型 | 外景 |
| 时间 | 清晨 |
| 情绪氛围 | 平静、怀旧、温馨 |
| 光线 | 晨曦金色侧逆光，晨雾 |

### 场景视觉Prompt

**整体街道**
```
写实摄影，超高清，临渊镇清晨街道
狭窄青石板路，湿润有露水
两侧古朴石木建筑，瓦片斑驳
清晨浓雾缭绕，能见度约20米
远处矿山黑色剪影隐约可见
简朴小店陆续开门
鸡犬相闻
古朴宁静氛围
负面：动漫，现代建筑，人群拥挤
```

**街道中景**
```
写实摄影，超高清，临渊镇街道中景
石板路延伸向远方
两侧旧式民居，木门石墙
晨雾在地面流动
早期阳光从雾中透出金色光芒
古朴宁静
负面：动漫，繁华热闹，霓虹灯
```

## S03 矿山入口（清晨）

| 属性 | 内容 |
|------|------|
| 场景类型 | 外景 |
| 时间 | 清晨 |
| 情绪氛围 | 紧张、压抑、命运转折 |
| 光线 | 灰蒙蒙晨光 |

### 场景视觉Prompt

**整体环境**
```
写实摄影，超高清，矿山入口全景
黑色岩石山体，矿洞深邃黑暗
矿道入口有简陋木支架
地面是碎石和泥土
多名童工排队等候，神情惶恐
管事穿黑色制服巡视
天空灰蒙蒙，晨雾缭绕
压抑紧张氛围
负面：动漫，明亮欢乐，绿色植被
```

**人群特写**
```
写实摄影，超高清，矿山入口人群
多名少年矿工排队
穿着破旧矿工服
神情紧张惶恐
低眉顺眼，不敢直视
管事在旁巡视
黑色制服，冷漠表情
灰蒙蒙天色
负面：动漫，开心表情，精致服装
