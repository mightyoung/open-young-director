# Seedance2-Storyboard-Generator 研究报告

## 项目概述

**GitHub**: https://github.com/liangdabiao/Seedance2-Storyboard-Generator

这是一个基于 **Claude Code + Seedance 2.0** 的 AI 视频制作工作流，专注于将小说、故事转化为多集视频系列。项目实现了从剧本创作到分镜脚本生成再到视频生成的完整流程。

### 核心工具链

| 工具 | 用途 |
|------|------|
| Claude Code | 剧本创作与分镜脚本生成 |
| Nana Banana Pro | 角色、场景、道具素材图生成 |
| Seedance 2.0 | 视频生成与延长 |

---

## 一、故事板结构

### 1.1 项目文件组织

```
写剧本和分镜/
├── [项目名]/
│   ├── [标题]_剧本.md           # 完整四幕结构剧本
│   ├── [标题]_素材清单.md       # 编号的素材生成提示词
│   └── [标题]_E[集数]_分镜.md   # 各集分镜脚本
```

### 1.2 素材编号规范

| 前缀 | 编号范围 | 类型 | 示例 |
|------|---------|------|------|
| C | C01-C99 | 角色（多角度） | C01 林冲·正面全身 |
| S | S01-S99 | 场景/地点 | S01 沧州草料场·雪景 |
| P | P01-P99 | 道具/物品 | P01 花枪 |
| FX | FX01-FX99 | 特效/氛围 | FX01 大雪纷飞 |

## 二、分镜脚本格式

每集分镜脚本包含三部分：

### 2.1 素材上传清单

```markdown
| 上传位置 | 素材ID | 素材描述 |
|---------|--------|----------|
| 图片1 | C01 | 林冲正面全身 |
| 图片2 | C02 | 林冲面部特写 |
| ...
```

### 2.2 Seedance Prompt（核心部分）

```
[风格描述]，[时长]，[画幅]，[氛围]

0-3秒：[镜头运动]，[场景建立]，[主体引入]
3-6秒：[镜头运动]，[情节发展]，[动作描述]
6-9秒：[镜头运动]，[高潮/冲突]，[情绪爆发]
9-12秒：[镜头运动]，[转折/过渡]
12-15秒：[镜头运动]，[结尾/落版]

【声音】配乐风格 + 音效 + 对白
【参考】@图片1 角色，@图片2 场景...
```

### 2.3 尾帧描述

记录本集最后一帧画面，用于下一集衔接：
- 主体状态
- 背景环境
- 光线色调
- 构图方式
- 氛围情绪

## 三、镜头分割策略

### 3.1 时间轴分段法

将 15 秒视频拆分为 **5 个 3 秒时段**：

| 时段 | 功能 | 内容 |
|------|------|------|
| 0-3s | 开场建立 | 高空俯拍，场景建立 |
| 3-6s | 发展引入 | 镜头推进，主体引入 |
| 6-9s | 情节发展 | 冲突展现，角色互动 |
| 9-12s | 高潮时刻 | 情感爆发，关键动作 |
| 12-15s | 结尾落版 | 氛围延续，画面定格 |

### 3.2 每时段三层描述

每个时段包含 **画面 + 镜头运动 + 音效** 三层信息：

```markdown
0-3秒画面：高空俯拍视角缓慢下降前推（推速约0.5倍），
从鸟瞰推至平视全景。被焚毁的山门废墟全景——
黑烟升腾，残垣断壁，碎裂匾额散落，主角孤独地站在废墟中央。
风声呼啸，余烬噼啪声。
```

## 四、视觉元素描述方法

### 4.1 风格统一前缀

所有素材提示词必须以统一风格前缀开头：

```markdown
2D中国动漫风格，类似《大护法》《雾山五行》，赛璐珞阴影处理，
线条清晰，色彩鲜明但沉稳，扁平化配色，电影感构图
```

### 4.2 角色描述要素

| 要素 | 描述内容 |
|------|----------|
| 外貌特征 | 头戴、身穿、腰系、脚蹬 |
| 表情神态 | 眼神、嘴角、整体气质 |
| 姿态动作 | 站姿、持物、动态 |
| 视觉标记 | 颜色方案区分不同角色 |

**示例**：
```markdown
林冲正面全身像，三十余岁，豹头环眼，燕颔虎须，头戴毡笠，
身穿鹦哥绿战袍，外罩羊皮袄，腰系鸾带，脚蹬豹皮靴，手持花枪，
威风凛凛的英雄姿态，漫天飞雪背景
```

### 4.3 场景描述要素

| 要素 | 描述内容 |
|------|----------|
| 地点环境 | 具体位置、环境特点 |
| 光线氛围 | 日景/夜景、光影效果 |
| 天气状况 | 雪景、雨景、晴天 |
| 构图特点 | 留白、景别、视角 |

**示例**：
```markdown
沧州草料场全景，大雪纷飞，成堆的草料被白雪覆盖，
几间草屋孤零零矗立在茫茫雪地中，天地一片苍茫，大面积留白
```

### 4.4 道具描述要素

| 要素 | 描述内容 |
|------|----------|
| 外观材质 | 形状、颜色、质感 |
| 功能用途 | 使用方式、摆放位置 |
| 细节特征 | 磨损、裂纹、装饰 |
| 光影效果 | 反光、阴影、照明 |

## 五、提示词模板

### 5.1 标准公式

```
【风格】_____风格，_____秒，_____比例，_____氛围

【时间轴】
0-X秒：[镜头] + [画面] + [动作] + [特效]
...

【声音】_____配乐 + _____音效 + _____对白

【参考】@图片1 _____，@视频1 _____
```

### 5.2 模板类型

| 模板类型 | 适用场景 | 时段分配 |
|----------|----------|----------|
| 叙事故事类 | 剧情短片 | 0-3s建立 / 3-7s发展 / 7-11s高潮 / 11-15s结尾 |
| 产品展示类 | 广告宣传 | 0-2s开场 / 2-5s展示 / 5-8s细节 / 8-15s落版 |
| 角色动作类 | 武打/舞蹈 | 0-3s亮相 / 3-6s起势 / 6-11s展示 / 11-15s定格 |
| 情感冲突类 | 情感戏 | 对白穿插、情绪递进 |
| 视频延长类 | 续接内容 | 无缝衔接、风格延续 |

## 六、运镜关键词

| 效果类型 | 关键词 |
|----------|--------|
| 推近 | 推镜头/缓慢推近/急速推进 |
| 拉远 | 拉镜头/逐渐拉远 |
| 摇摄 | 左摇/右摇/横移 |
| 跟随 | 跟随镜头/跟拍 |
| 环绕 | 环绕镜头/360度旋转 |
| 升降 | 升镜头/降镜头/俯冲 |
| 特效 | 希区柯克变焦/一镜到底 |
| 手持 | 手持晃动 |

### 高级运镜技巧

- **希区柯克变焦**：镜头前推同时焦距拉远，人物大小不变但背景压缩
- **一镜到底**：不切镜连续跟拍，增加临场感
- **360度环绕**：以角色为中心旋转一圈
- **时间冻结环绕**：画面定格后镜头绕行展示细节
- **第一人称主观视角**：从角色眼睛看出去的画面

## 七、多模态引用语法

Seedance 2.0 支持通过 `@` 符号引用素材：

```markdown
@图片1 作为首帧
@图片2 作为尾帧
@图片3 作为角色形象参考
@图片4-6 作为场景参考
@视频1 参考运镜方式
@视频2 参考动作节奏
@音频1 用于配乐
```

### 素材限制

- 最多 12 个文件（图片 ≤9 + 视频 ≤3 + 音频 ≤3）
- 视频/音频参考总时长 ≤15 秒

---

## 八、视频延长技术

从第 2 集开始使用视频延长功能保持连续性：

```
将@视频1延长15s（生成长度选择15秒）
```

### 串联链

```
片头(新生成) → E1(新生成) → E2(延长E1) → E3(延长E2) → ... → E20(延长E19)
```

### 转场处理

当需要场景切换时（如夜景转白天），在提示词中显式添加转场指令：

```markdown
画面渐暗过渡，天色从暴雨夜晚转为雨后月夜
```

## 九、关键发现总结

### 9.1 核心创新

1. **四幕结构剧本**：起承转合，每幕5集，每集有明确情绪基调
2. **素材编号系统**：C/S/P/FX 四类编号，支持 @语法灵活引用
3. **时间轴分段法**：15秒拆5段，每段画面+镜头+音效三层描述
4. **视频延长串联**：通过延长功能实现多集无缝衔接
5. **风格统一前缀**：保证跨集视觉一致性

### 9.2 视觉描述要点

1. **角色**：外貌+表情+姿态+颜色方案（区分度）
2. **场景**：环境+光线+天气+构图（沉浸感）
3. **道具**：材质+功能+细节+光影（真实感）
4. **氛围**：光影+色调+质感+情绪（感染力）

### 9.3 局限性

1. 敏感词机制不透明，排查困难
2. 长提示词（300+字）指令遵循不稳定
3. 视频编辑能力有限，无法精确修改某几秒
4. 写实真人脸部素材暂不支持
