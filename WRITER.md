# 小说创作宪法 (WRITER.md)

本文件定义的准则具有最高优先级，所有创作 Agent 必须无条件遵守。

## 1. 语言审美准则 (Stylistic Commandments)
- **去AI化**：严禁使用“极其、非常、仿佛、似乎、某种、令人惊讶的是”等垫话。
- **物理化**：情感必须通过具体的物理反馈（心跳、冷汗、手指颤抖）来表达，严禁直接描述情感名词。
- **禁词表**：[突然, 居然, 竟然, 恐怖如斯, 倒吸一口冷气]。

## 2. 世界观硬约束 (World-building Constraints)
- **能量守恒**：任何超自然力量的动用必须有明确的体力、精神力或资源消耗。
- **地理逻辑**：角色移动必须符合 `character_gps` 记录，严禁瞬间移动。
- **战力天平**：低境界战胜高境界必须描写极其惨烈的代价或利用了规则漏洞。

## 3. 角色塑造准则 (Character Guidelines)
- **动机先行**：角色做的每一个决定必须符合其 `hidden_agenda`。
- **拒绝降智**：反派必须拥有与主角对等的智商和生存逻辑。

## 4. 叙事技巧 (Narrative Techniques)
- **伏笔闭环**：每三章必须收割或升格至少一个来自 `seeds_registry` 的细节。
- **一张一弛**：根据 `pacing_state` 调整张力，连续高潮后必须有喘息。
