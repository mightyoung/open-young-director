# Seedance Prompt Skill 研究报告

> 来源：https://github.com/songguoxs/seedance-prompt-skill
> 分析日期：2026-03-29

## 概述

Seedance Prompt Skill 是为字节跳动 **Seedance 2.0（即梦）** 视频生成平台设计的 Claude Code 技能，可将自然语言描述转换为结构化中文视频提示词。

---

## 一、十大核心提示词生成能力

| 编号 | 能力 | 说明 |
|------|------|------|
| 1 | 纯文本生成 | 无需参考素材，纯文字生成视频 |
| 2 | 一致性控制 | 通过参考图保持角色/产品/场景统一 |
| 3 | 运镜与动作复刻 | 复刻参考视频的镜头语言和动作 |
| 4 | 创意模板/特效复刻 | 模仿转场、广告、电影特效 |
| 5 | 剧情创作/补全 | 从分镜图或脚本自动生成剧情 |
| 6 | 视频延长 | 对已有视频前后向平滑延长 |
| 7 | 声音控制 | 音色参考、对白生成、音效设计 |
| 8 | 一镜到底 | 连续长镜头不切换 |
| 9 | 视频编辑 | 角色替换、剧情颠覆、元素增减 |
| 10 | 音乐卡点 | 画面节奏与音乐节拍精准同步 |

## 二、提示词模板结构

### 2.1 基础提示词公式

```
(主体描述) + (动作序列) + (环境/光影) + (镜头语言) + (风格关键词)
```

### 2.2 时间戳分镜法（核心技巧）

对于 15 秒视频，按秒精确控制每个镜头：

```
15秒[主题]，[色调]，0-3秒：[画面+镜头]；
4-8秒：[画面+镜头]；
9-12秒：[画面+镜头]；
13-15秒：[画面+镜头]，[音效]。
```

**完整示例：**
```
15秒仙侠高燃战斗镜头，金红暖色调，0-3秒：低角度特写主角蓝袍衣摆被热浪吹得猎猎飘动，双手紧握雷纹巨剑，剑刃赤红电光持续爆闪，地面熔岩翻涌冒泡，远处魔兵嘶吼着冲锋逼近，主角低喝"今日，便以这柄剑，镇尔等邪祟！"，伴随剑鸣与熔岩咕嘟声；4-8秒：环绕摇镜快切，主角旋身挥剑，剑刃撕裂空气迸射红色冲击波，前排魔兵被击飞碎裂成灰烬，伴随剑气破空声与魔兵惨嚎；9-12秒：仰拍拉远定格慢放，主角跃起腾空，剑刃凝聚巨型雷光电弧劈向魔兵群；13-15秒：缓推特写主角落地收剑的姿态，衣摆余波微动，冷声道"此界之门，不容踏越"，音效收束为余音震颤与渐弱风声。
```

### 2.3 技术参数指定法

在提示词开头声明技术规格：

```
[尺寸]竖屏/横屏 + [画幅比]2.35:1/16:9/9:16 + [帧率]24fps + [时长]Xs + [色调/风格总纲]
```

**示例：**
```
2.35:1，24fps，15秒，8镜头硬切，霓虹高饱和冷暖对比，现代舞台，浅景深突出动作，动作清晰，运动模糊真实，声音设计优先：舞步声、鞋底摩擦、呼吸、衣料声必须清晰并与节拍贴合，禁止文字logo水印
```

### 2.4 多模态参考模式

```
[角色]@图片N + [动作/剧情描述] + [场景]@图片N + [运镜/光影]
```

## 五、场景化提示词策略

### 5.1 电商/广告

- 产品360度旋转展示、爆炸分解、3D渲染特效
- 第一人称沉浸式手作体验
- 配合广告词和品牌logo

**示例：**
```
@图片1中的可口可乐饮料，360度高速旋转2圈后，突然停住蓄力分裂成了3个部分进行展示。随后分解后的可口可乐饮料罐的上中下三部分快速向内旋转合成，一罐完整的可口可乐饮料，3D渲染产品展示特效，动感产品特效展示
```

### 5.2 仙侠/奇幻

- 使用首尾帧控制变身/变装效果
- 时间戳分镜法控制每段画面
- 详细特效描述（法阵、能量波、粒子效果）
- 台词用引号标注，指定语气

### 5.3 短剧/对白

- 画面+台词分开描述
- 台词标注角色和情绪
- 音效单独描述
- 精准时长控制

**示例：**
```
画面（0-5秒）：特写女主撕契约镜头，纸屑飘落，总裁单膝跪地伸手阻拦，眼神慌乱，女主侧身躲开，嘴角挂着冷漠笑意
台词1（总裁，卑微慌乱）：苏晚！契约还没结束，你不能走！我给你钱，给你地位！
画面（6-10秒）：女主抬脚避开他的手，将撕碎的契约纸扔在他脸上，镜头扫过周围宾客的窃窃私语
台词2（女主，冷漠反杀）：契约？顾总，当初是你说，我连给你提鞋都不配，现在求我？晚了！
画面（11-15秒）：总裁僵在原地，脸上沾着纸屑，女主转身昂首离开，红裙裙摆飘动
音效：华丽又带张力的背景音，契约撕碎的声响，宾客轻微的窃窃私语声
```

### 5.4 科普教学

- 4K医学CGI风格
- 半透明人体结构可视化
- 科学过渡自然流畅
- 配合教育性旁白

### 5.5 MV/音乐卡点

- 指定画幅比（2.35:1）和帧率（24fps）
- 分镜头描述每个镜头的场景、动作、音效
- 强调声音设计与节拍同步
- 多图卡点参考视频节奏

## 六、超长视频分段策略（>15秒）

### 6.1 分段规则

1. 将总时长按叙事节奏切分为多个片段，每段≤15秒
2. 每段之间必须有**画面衔接点**：上一段结尾状态 = 下一段开始状态
3. 第一段正常生成，后续每段使用「将@视频1延长Xs」格式

### 6.2 输出格式

```
## 超长视频提示词（总时长约Xs）

**主题**：[一句话概括]
**总段数**：[N段]
**建议比例**：[16:9 / 9:16 / 1:1]

---

### 第1段（0-15秒）—— 正常生成

**生成时长**：15秒

#### 提示词

[完整提示词，时间戳分镜]

#### 衔接点

本段结尾画面：[精确描述结尾画面状态]

---

### 第2段（15-30秒）—— 视频延长

**操作**：将第1段生成的视频上传为@视频1
**生成时长**：15秒

#### 提示词

将@视频1延长15秒。[接续内容的时间戳分镜描述]
```

### 6.3 分段时长建议

| 总时长 | 推荐分段 |
|--------|----------|
| 16-30秒 | 2段（首段15秒 + 延长段） |
| 31-45秒 | 3段 |
| 46-60秒 | 4段 |
| >60秒 | 建议拆分为独立场景分别生成 |

## 九、最佳实践总结

1. **描述具体有画面感** — "穿红风衣的女人走过雨后霓虹闪烁的街道"远比"一个女人在走路"效果好
2. **13-15秒视频使用时间戳分镜** — 按秒划分镜头，精确控制每段画面
3. **台词和音效单独描述** — 与画面描述分行书写，结构更清晰
4. **结尾添加禁止项** — 排除水印、字幕、Logo等
5. **参考图片风格要匹配主题** — 古风用水墨风格，赛博朋克用科幻CG
6. **台词用引号包裹** — 并标注角色名称和情绪
7. **不要忽略情绪氛围描述** — 对最终效果影响很大

## 十、平台规格

| 维度 | 规格 |
|------|------|
| 图片输入 | jpeg/png/webp/bmp/tiff/gif，最多9张，单张<30MB |
| 视频输入 | mp4/mov，最多3个，总时长2-15秒，单个<50MB |
| 音频输入 | mp3/wav，最多3个，总时长≤15秒 |
| 文件上限 | 最多12个文件（图片+视频+音频合计） |
| 生成时长 | 4-15秒/次 |
| 分辨率 | 支持2K输出 |

> 注意：平台不支持上传含有写实真人脸部的素材，系统会自动拦截。
