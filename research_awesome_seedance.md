# Awesome Seedance 2.0 研究报告

## 概述

Awesome Seedance 2.0 是一个关于 Seedance（即梦）视频生成 AI 的优质提示词精选仓库，收集自 X (Twitter)、微信及顶级提示词工程师的实战经验。

---

## 一、视频提示词结构

### 1.1 标准模板结构

```
风格：<电影风格描述>
时长：<X秒>

[时间区间] 镜头 X：<景别/视角>
<场景详细描述>
<动作和情感描写>
<对白提示（可选）>
```

### 1.2 时间分镜格式

使用时间码标记镜头段落：
- `[00-05s]` - 0到5秒
- `[05-10s]` - 5到10秒
- `[10-15s]` - 10到15秒

每个段落内包含：
1. **景别/视角定义**：如 `特写`、`远景动作`、`驾驶舱视角`
2. **场景描述**：环境、光线、氛围
3. **角色动作**：具体行为和表情
4. **对白/字幕提示**：用于控制口型和字幕

### 1.3 分层描述法

采用**远-中-近-特**的分层结构：

| 层级 | 描述 | 示例 |
|------|------|------|
| 远景 | 交代环境、规模感 | `极远景（规模感）` |
| 中景 | 主体与背景关系 | `双人中景` |
| 近景 | 情感表达 | `过肩视角 (OTS)` |
| 特写 | 细节、微妙情绪 | `极端特写与微表情` |

## 二、摄影技巧

### 2.1 镜头类型术语

| 术语 | 含义 | 适用场景 |
|------|------|----------|
| 特写 | Close-up, 聚焦角色面部或细节 | 情感表达 |
| 远景 | Long shot, 展现整体环境 | 建立场景 |
| 极远景 | Extreme long shot | 史诗规模 |
| 内景/特写 | Interior close-up | 车内、屋内 |
| 驾驶舱视角 | POV from cockpit/driver | 紧张感 |
| 过肩视角 (OTS) | Over-the-shoulder | 对话场景 |

### 2.2 运镜技巧

```
镜头切换 - 不同角度的切换组合
镜头剧烈晃动 - 模拟手持摄影、恐慌场景
镜头跟随 - 跟随角色或物体运动
跳跃剪辑 - 快速切入高潮
```

### 2.3 视觉效果参数

| 效果 | 描述 | 应用场景 |
|------|------|----------|
| 运动模糊 | 高速运动时的速度感 | 赛车、动作 |
| 慢动作 | 关键时刻的强调 | 高潮、情感爆发 |
| 抽帧效果 | 定格动画感 | 艺术化表达 |
| 浅景深 | 虚化背景，突出主体 | 人像、特写 |

## 三、角色描述方法

### 3.1 外貌描写公式

```
<角色定位>（<服装>，<发型>，<特殊细节>）
```

**示例：**
- `老赛车手（戴头盔）看过来，冷静而专注`
- `穿着卡其色风衣的男人紧紧握着听筒`
- `深情的霸道总裁男主（黑色大衣，湿漉漉的头发，红了眼眶）`

### 3.2 情感描写层次

1. **外在表现**：
   - 眼神：空洞却饱含深情、冷静而专注、眼神中充满了肾上腺素
   - 表情：翻白眼、嘴唇轻微颤抖
   - 肢体：呼吸沉重、紧握方向盘

2. **内在状态**：
   - 通过动作暗示：`他微微点头，低声说`
   - 通过环境烘托：`眼神死鱼眼，毒舌`

### 3.3 对白与口型控制

```
对白提示：他微微点头，低声说："出发吧。"
对白情感映射：展现出"想触碰却收回手"的极致克制与孤独。
```

## 六、关键发现与最佳实践

### 6.1 成功的提示词特征

1. **时间轴清晰**：明确的 `[00-05s]` 分段
2. **景别明确**：每个镜头标注景别类型
3. **细节丰富**：动作、表情、环境光缺一不可
4. **情感锚点**：核心对白或情感基调明确

### 6.2 高级技巧

| 技巧 | 效果 |
|------|------|
| 动态模糊转化为光轨 | 速度感 |
| 慢动作定格 + 切黑 | 电影感高潮 |
| 镜面反射反转 | 悬疑/超现实 |
| 抽帧 + 动态模糊叠加 | 艺术化表达 |
| 口型与字幕逐字一致 | 专业度提升 |

### 6.3 常见问题规避

- 避免模糊描述：使用具体动作而非"动了一下"
- 避免过长镜头：每段 5 秒左右为宜
- 避免复杂场景：Seedance 擅长特定物体的动作而非复杂交互

## 八、结论

Seedance 2.0 提示词的核心在于：

1. **结构化**：使用时间分 `[时间]` + 景别 `镜头 X：景别` 的标准格式
2. **细节化**：动作、表情、环境光缺一不可
3. **情感化**：通过核心对白或导演笔记传达情感基调
4. **技术化**：明确标注风格、时长、技术参数

掌握以上格式和技巧，可显著提升 Seedance 2.0 视频生成的质量和一致性。
