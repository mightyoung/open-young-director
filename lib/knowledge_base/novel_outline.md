# 第一部分：感情线设计

## 核心三角关系

```
        柳如烟（反派系）
           ↓ 退婚
    韩林 ←→ 萧婉儿（正女主）
           ↑ 救命之恩
         叶尘（情敌）
```

## 女主角详细设定

### 萧婉儿（正女主）
| 项目 | 内容 |
|------|------|
| **身份** | 丹师协会天才炼丹师，萧家嫡女 |
| **外貌** | 身着一袭白衣，清丽脱俗，眉心有一点朱砂 |
| **性格** | 外冷内热、善良坚韧、有点傲娇 |
| **特长** | 炼丹天赋绝伦，擅长火系功法 |
| **与主角关系** | 秘境中相遇，韩林救她一命，后成为挚友兼暧昧对象 |
| **感情线** | 日久生情，从感激到爱慕，多次暗示，被韩林因复仇心切而忽略 |
| **结局** | 与韩林终成道侣，共赴巅峰 |

### 柳如烟（反派系/工具人）
| 项目 | 内容 |
|------|------|
| **身份** | 太虚宗附近柳家之女，后嫁给叶尘 |
| **外貌** | 容貌清丽但眼神冷漠 |
| **性格** | 野心勃勃、嫌贫爱富、表面温婉内心算计 |
| **与主角关系** | 韩林前未婚妻，退婚推动者，后悔者 |
| **感情线** | 曾与韩林有婚约，退婚时嘲讽"废物"，后期求复合被拒 |
| **结局** | 被叶尘抛弃，跪求韩林被拒，最终孤独终老 |

### 慕容雪（隐藏女主）
| 项目 | 内容 |
|------|------|
| **身份** | 天机阁少阁主，神秘势力 |
| **外貌** | 银发紫眸，气质出尘如仙 |
| **性格** | 冷静睿智、算无遗策、对韩林有特殊感觉 |
| **特长** | 擅长推演天机，情报网络遍布大陆 |
| **与主角关系** | 早期给予帮助，多次神秘出现，后期成为重要盟友 |

| **结局** | 等待揭示，可能单箭头或开放结局 |

## 主要人物卡片

### 韩林（主角）
- **身份**: 太虚宗弃徒 → 太古魔帝传人 → 太古盟主
- **外貌**: 剑眉星目，初期略显瘦弱，传承后气质凌厉
- **性格**: 沉稳内敛、重情重义、坚韧不拔、爱恨分明
- **背景**: 寒门之子，父亲为救宗门而死，母亲早亡
- **核心冲突**: 复仇 vs 保护爱人、仇恨 vs 新生
- **人物弧光**: 废物 → 强者 → 领袖 → 守护者

### 萧婉儿
- **身份**: 丹师协会天才、萧家嫡女
- **外貌**: 白衣胜雪，眉心朱砂，清丽绝伦
- **性格**: 外冷内热、傲娇、善良、坚定
- **背景**: 丹师世家，自幼展现炼丹天赋，被家族寄予厚望
- **核心冲突**: 家族使命 vs 个人感情
- **人物弧光**: 孤立 → 信任韩林 → 勇敢追爱

### 柳如烟
- **身份**: 柳家之女 → 叶家少夫人
- **外貌**: 容貌清丽，眼神冷漠
- **性格**: 野心勃勃、嫌贫爱富、表面温婉
- **背景**: 小家族之女，欲借婚姻改变命运
- **核心冲突**: 野心 vs 现实的绝望
- **人物弧光**: 得势 → 失势 → 求复合被拒 → 疯狂 → 毁灭

### 叶尘
- **身份**: 太虚宗内门天才 → 万魔窟客卿
- **外貌**: 英俊潇洒，但眼神阴鸷
- **性格**: 阴险狡诈、嫉妒心强、善于伪装
- **背景**: 宗门长老之子，自幼被捧杀，目中无人
- **核心冲突**: 自负 vs 被韩林击败
- **人物弧光**: 狂妄 → 被击败 → 投靠魔窟 → 彻底覆灭

### 慕容雪
- **身份**: 天机阁少阁主
- **外貌**: 银发紫眸，气质如仙
- **性格**: 冷静睿智、算无遗策、神秘莫测
- **背景**: 天机阁上任阁主收养，对韩林有前世记忆（悬念）
- **核心冲突**: 命运推演 vs 个人情感
- **人物弧光**: 神秘旁观 → 主动介入 → 揭示真相

### 莫老
- **身份**: 秘境守护者、太古魔帝残魂
- **外貌**: 白发苍苍，虚幻身影
- **性格**: 睿智、慈祥、有点玩世不恭
- **背景**: 太古魔帝的一缕残魂，守护传承
- **核心冲突**: 传承责任 vs 等待有缘人
- **人物弧光**: 等待 → 认可韩林 → 传承完成 → 消散

### 小瓶灵
- **身份**: 神秘器灵
- **外貌**: 圆润可爱的小瓶子形态，会变换表情
- **性格**: 贪吃、傲娇、忠诚
- **背景**: 上古至宝器灵，拥有时间法则
- **核心冲突**: 贪吃属性 vs 辅助修炼
- **人物弧光**: 傲娇嫌弃 → 真香 → 生死与共
