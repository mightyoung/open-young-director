# 第一章：废物少年 - Seedance 2.0 视频提示词

## 场景1：退婚羞辱

**【绝对主体与物理动势】**
韩林：二十岁左右，洗得发白的青色长袍，黑色短发，眼神深沉如古井，沉默坚毅，脊背挺直
柳如烟：二十岁左右，一袭素白长裙如仙子临尘，青丝微扬，肌肤胜雪，目光清冷如霜雪，神情冷漠决绝
动作轨迹：韩林静立观望 → 柳如烟缓步向前 → 玉佩碎裂 → 二人对视无言

**【环境场与情绪光影】**
场景：太虚宗演武场，清晨薄雾缭绕，测灵台矗立中央，四周人群攒动
光线：金色晨光穿透薄雾形成体积光，冷白色轮廓光勾勒人物
氛围：肃穆，张力、羞辱与隐忍交织，蓄势待发

**【光学与摄影机调度】**
开始帧：航拍全景，镜头缓慢下降
关键帧：中景跟随 → 特写手部 → 玉佩碎裂慢动作 → 双人对视中景 → 过肩韩林特写
景别：全景 → 中景 → 特写 → 双人中景 → 面容特写
焦段：35mm电影镜头 → 50mm标准镜头 → 90mm微距
运动：稳定下降 → 缓慢推进 → 慢动作 → 稳定横移 → 缓慢推近

**【时间轴与状态演变】**
0-3秒：航拍全景，晨光薄雾，镜头下降聚焦人群
3-5秒：镜头锁定演武场边缘，韩林独自站立静望
5-8秒：柳如烟素白长裙缓步向前，镜头跟随推进
8-10秒：玉佩坠落碎裂，灵光四散如星屑，慢动作
10-12秒：二人相隔数步无语对视，晨光侧逆光
12-14秒：韩林眉头紧锁目光平静，脊背挺直
14-15秒：特写手部五指收紧，淡出

**【美学介质与底层渲染参数】**
质感：高度写实风格
渲染：8K超清，电影级画质，精细渲染
其他：无模糊，无水印，电影感构图

## 场景2：取出令牌

**【绝对主体与物理动势】**
韩林：眉头紧锁，眼神坚定，手握青玉令牌
动作轨迹：右手探入怀中 → 取出令牌 → 拇指摩挲 → 五指收紧蓄力

**【环境场与情绪光影】**
场景：测灵台前，晨光初照，薄雾缭绕，青石地面，古朴庄严
光线：暖金色晨光聚焦令牌，体积光穿透薄雾
氛围：悲壮、决心、传承与牺牲

**【光学与摄影机调度】**
开始帧：中景测灵台前，晨光侧打
关键帧：手部特写探入怀中 → 令牌上举微距 → 握拳收紧特写 → 拇指摩挲眼神 → 拉远至全身
景别：中景 → 特写 → 微距 → 面容特写 → 全景
焦段：50mm标准镜头 → 90mm微距镜头
运动：稳定横移 → 缓慢推近 → 定格 → 缓慢拉远

**【时间轴与状态演变】**

4-6秒：青玉令牌缓缓上举特写，正面刻有"韩"字
6-8秒：拇指摩挲玉面，眼神闪过回忆与决心
8-10秒：五指收紧指节发白，握拳蓄力
10-12秒：镜头拉远至中景，晨光笼罩全身如圣迹
