# 《太古魔帝传》项目约束

> 本文件是《太古魔帝传》小说创作的核心约束，所有创作必须遵守。

---

## 一、字数约束

### 每卷目标

| 卷名 | 卷名(英) | 目标字数 | 章节数 | 每章平均字数 |
|------|----------|----------|--------|--------------|
| 第一卷：废物崛起 | Volume 1 | **300,000字** | 60章 | ~5,000字/章 |
| 第二卷：筑基之路 | Volume 2 | **300,000字** | 60章 | ~5,000字/章 |
| 第三卷：问道巅峰 | Volume 3 | **300,000字** | 60章 | ~5,000字/章 |
| 第四卷：逆仙成帝 | Volume 4 | **300,000字** | 60章 | ~5,000字/章 |

### 总目标

**全书总字数目标：1,200,000字（120万字）**

### 章节字数标准

| 章节类型 | 字数范围 | 说明 |
|----------|----------|------|
| 日常/修炼章节 | 4,500-5,500字 | 铺垫剧情，描写修炼 |
| 战斗/高潮章节 | 5,500-7,000字 | 核心打斗场面，详细描写 |
| 感情/文戏章节 | 4,000-5,000字 | 感情戏，内心独白 |
| 转折/揭秘章节 | 5,000-6,000字 | 关键剧情点 |
| 卷末高潮章节 | 6,000-8,000字 | 每卷最终决战 |

## 三、结构约束

### 每卷结构（60章）

```
第1-5章   ：序章/屈辱篇
第6-15章  ：觉醒篇
第16-25章 ：修炼篇
第26-35章 ：崛起篇
第36-45章 ：危机篇
第46-55章 ：反击篇
第56-60章 ：卷末高潮/结局篇
```

### 每卷必备要素

| 要素 | 说明 |
|------|------|
| 开局屈辱 | 主角遭受屈辱，激发斗志 |
| 金手指觉醒 | 逆仙录/逆天灵根展现 |
| 三年之约 | 每卷至少一个约定/誓言 |
| 势力斗争 | 穿插多方势力博弈 |
| 感情线 | 至少一条明确感情线 |
| 爽点密集 | 每3-5章一个小爽点 |
| 大高潮 | 每卷至少2-3个大高潮 |
| 伏笔埋设 | 每卷埋3-5个伏笔供后续回收 |
| 悬念结尾 | 每卷结尾留有悬念 |

## 六、爽点设计约束

### 爽点密度要求

| 爽点类型 | 间隔要求 | 示例 |
|----------|----------|------|
| 小爽点 | 每2-3章 | 击败欺压者、修炼突破 |
| 中爽点 | 每5-8章 | 越级击败敌人、揭露阴谋 |
| 大爽点 | 每15-20章 | 三年之约完成、击杀BOSS |

### 必备经典爽点

- [ ] 废物逆袭：主角从被踩到踩人
- [ ] 退婚反转：退婚是深情守护
- [ ] 三年之约：经典契约热血复仇
- [ ] 越级挑战：炼气击败筑基
- [ ] 打脸众人：让嘲笑者闭嘴
- [ ] 师父认可：获得大能认可
- [ ] 美人倾心：多位美人好感
- [ ] 复仇成功：完成三年/多年夙愿

## 七、伏笔约束

### 每卷伏笔数量

| 类型 | 数量 | 说明 |
|------|------|------|
| 本卷回收伏笔 | 3-5个 | 本卷内回收 |
| 跨卷伏笔 | 3-5个 | 下一卷回收 |
| 长期伏笔 | 2-3个 | 贯穿全书 |

### 伏笔记录要求

每卷大纲必须包含伏笔表格，格式如下：

```markdown
| 伏笔 | 埋入章节 | 回收章节 | 效果 |
|------|----------|----------|------|
| XXX | ChN | ChN+M | 读者恍然大悟 |
```

## 八、质量约束

### 内容质量

- [ ] 每章必须有明确的目标和悬念
- [ ] 战斗场面必须有层次感（试探→激烈→危机→反转→胜利）
- [ ] 感情戏必须有细腻的心理描写
- [ ] 反派必须有智商，不能无脑送
- [ ] 主角必须有成长曲线，不能一直无敌

### 发布前检查

- [ ] 字数是否达标（≥3,500字）
- [ ] 是否有错别字
- [ ] 是否符合当前修为设定
- [ ] 伏笔是否对应
- [ ] 角色名是否一致
