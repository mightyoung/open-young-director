## 三、金丹→化神 放缓详细方案

### 3.1 当前问题章节（ch12-20）

已生成章节的境界进度过快，需要大幅修订。

**修订策略**：
- ch12: 金丹初期 → 筑基后期（降低一个大境界）
- ch15: 筑基圆满 → 筑基大圆满（维持筑基期）
- ch18: 金丹后期 → 金丹初期（降低两个小境界）

### 3.2 金丹期各小境界详细描述

#### 金丹初期（ch121-140，约7.5万字）

**特征**：
- 丹成极品，金丹光华内敛
- 寿元增至500年
- 神识覆盖百里

**需要描写的要素**：

## 四、内容丰富化方案

### 4.1 每章最低字数标准

| 类型 | 原最低 | 现最低 | 说明 |
|------|--------|--------|------|
| 普通章节 | 2000字 | 4000字 | 增加对话和心理描写 |
| 战斗章节 | 3000字 | 5000字 | 增加招式描写和场面 |
| 突破章节 | 4000字 | 6000字 | 增加细节和感悟 |

### 4.2 每章必要元素清单

```
□ 场景描写（至少2处自然/建筑环境）
□ 人物对话（至少3段，每段不少于20字）
□ 心理活动（主角至少1段内心独白）
□ 修炼/战斗细节（具体招式、感受）
□ 伏笔/暗示（下章或后续情节）
□ 配角反应（至少1个配角的视角）
```

### 4.3 境界突破标准流程（每突破一个小境界都需要）

1. **瓶颈出现**（1章）：修炼遇到困难
2. **寻找方法**（1章）：查阅典籍、请教他人、探索遗迹
3. **获得机缘**（1章）：得到丹药、材料、功法、指点
4. **闭关突破**（1章）：详细描写突破过程
5. **境界巩固**（1章）：适应新境界，展现新能力

**最少5章/次突破**，按金丹期4个小境界计算 = 最少20章

## 七、总结

### 7.1 核心原则

1. **每大境界至少20章**：金丹、元婴、化神各需要足够的篇幅
2. **每小境界至少5章**：瓶颈→机缘→突破→巩固
3. **每章4000字以上**：确保内容充实
4. **境界压制原则**：除非特殊情况，主角境界不得高于对手太多

### 7.2 修复优先级

1. **最高优先级**：修改 ch012-ch020 的境界设定
2. **高优先级**：重新生成 ch009-ch011
3. **中优先级**：制定 ch021-ch060 的详细大纲
4. **普通优先级**：后续章节按新节奏生成
