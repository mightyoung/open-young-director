# 《太古魔帝传》三线融合大纲·第一卷

> **架构**：表·里·源 三层结构
> **篇幅**：ch001-ch060，约30万字
> **目标**：慢热积累，境界放缓

---

## 一、核心架构

### 1.1 三层结构定义

```
┌─────────────────────────────────────────────────────────┐
│ 【表层】宗门修仙线                                         │
│  - 主角身份：太虚宗外门弟子                                │
│  - 主线剧情：废物逆袭、三年之约、宗门大比                   │
│  - 功法：逆仙录（太虚子传授，正道功法）                     │
│  - 境界：炼气期1-9层（60章）                              │
│  - 女主：柳如烟                                           │
│  - 宿敌：叶尘                                             │
└─────────────────────────────────────────────────────────┘
                          ↓ 暗线发展
┌─────────────────────────────────────────────────────────┐
│ 【里层】魔帝觉醒线                                         │
│  - 传承来源：体内封印魔帝残魂                               │
│  - 功法：太古魔经（隐秘传授，梦境/危机时觉醒）             │
│  - 觉醒度：ch1封印 → ch30觉醒5% → ch60觉醒20%            │
│  - 守护者：魔帝残魂（沉睡中，低语提示）                    │
│  - 使用代价：魔化风险、记忆模糊、神识混乱                   │
└─────────────────────────────────────────────────────────┘
                          ↓ 未来预告
┌─────────────────────────────────────────────────────────┐
│ 【源层】记忆碎片                                           │
│  - 性质：魔帝传承中的"未来可能性"                          │
│  - 触发：突破瓶颈、接触魔道物品、梦境中                     │
│  - 内容：已生成ch012-ch020的高境界战斗画面                  │
│  - 作用：悬念埋设、伏笔回收、读者期待                      │
└─────────────────────────────────────────────────────────┘
```

### 1.2 境界进度表（第一卷 ch001-ch060）

| 阶段 | 章节 | 境界 | 核心目标 |
|:-----|:-----|:-----|:---------|
| 第一阶段 | ch001-010 | 炼气1-3层 | 觉醒、逆袭初露 |
| 第二阶段 | ch011-025 | 炼气4-6层 | 宗门大比、叶家阴谋 |
| 第三阶段 | ch026-040 | 炼气7-8层 | 三年之约、升仙大会 |
| 第四阶段 | ch041-055 | 炼气9层 | 大战余波、筑基前奏 |
| 第五阶段 | ch056-060 | 炼气9层→圆满 | 冲击筑基、卷末高潮 |

### 1.3 双功法体系

```
【表】逆仙录（正道功法）
  - 第一重·逆气：炼气期（ch1-60）
  - 第二重·逆脉：筑基期（ch61-120）
  - 第三重·逆血：金丹期（ch121-180）
  - 特点：稳扎稳打，每重需要大量积累

【里】太古魔经（魔道功法）
  - 觉醒度：0% → 5% → 20%
  - 使用场景：梦境、无意识、生死危机
  - 代价：魔化、记忆缺失、神识损伤
  - 进阶条件：与逆仙录同修，相互促进
```

```
  - 表现：魔气外泄、攻击力暴增
  - 持续：数小时
  - 代价：神识损伤、情绪暴躁

【严重失控（魔化）】
  - 表现：完全失控，敌我不分
  - 触发：过度使用或情绪极端
  - 代价：可能失去自我
```
