# 作品简介

## 《太古魔帝传》

> 废物逆袭青云上，魔帝传承乱苍穹！
> 三年之约，看我以凡躯撼仙神；万魔窟前，问谁与我共赴生死！

---

## 一句话简介

太虚宗废物少年韩林，测灵大典受尽屈辱，退婚之辱、父亲冤死、宗门阴谋接踵而至。绝望之际，体内魔帝残魂觉醒——太古魔经、逆仙录，双道同修，逆天改命！三年之约到期日，便是废物逆袭时。

---

## 世界观简介

太虚宗，修仙界十大正道宗门之一，坐落于东荒大陆太虚山脉。

修仙境界分为：炼气（1-9层+圆满）、筑基、金丹、元婴、化神……每一步皆是天堑。

然修仙界并非表面祥和——天玄宗图谋一统修仙界，鬼冥宗暗中蛰伏，血河老祖的魔爪延伸至各大宗门。上古时期的恩怨，正在悄然复苏。

---

## 主线简介

### 表层·宗门修仙线

太虚宗外门，废物少年韩林丹田破碎，受尽欺凌。

测灵大典之上，伪灵根的判定将他打入深渊。未婚妻柳如烟当众退婚，践踏他最后的尊严。殊不知，那一纸退婚书，并非嫌弃，而是她拼死换来的保护——她身怀地阶上品灵根，却被家族逼婚，她知道韩林父亲死因可疑，知道有人在暗中窥伺。

为了让韩林远离漩涡，她选择用最决绝的方式：当众退婚，让他恨她，让他离开这是非之地。

韩林撕毁退婚书，立下三年之约：三年后，他会上门娶她，让她为今日之辱付出代价！

### 里层·魔帝觉醒线

然而，命运在此刻逆转。

韩林父亲遗留的逆仙录，令他脱胎换骨；而在梦境深处，一缕魔帝残魂正在苏醒。那是太古魔帝的第九世轮回身，体内封印着上古魔帝的记忆与力量。

魔帝残魂沉睡于韩林识海深处，每当生死危机、修炼瓶颈，便会低语警示、梦中传法。太古魔经，隐秘传授；逆天灵根，逆骨觉醒。双功法同修——逆仙录主正道根基，太古魔经掌魔道杀伐，相辅相成。

但魔道之力，非无偿之物。使用魔气，轻则毛发变黑、眼睛泛红；重则神识损伤、记忆模糊；极端失控，则敌我不分、丧失神智。

### 源层·记忆碎片

随着魔帝觉醒度提升，记忆碎片开始浮现——那是来自"未来"的画面：万魔窟中金蛋破碎、化神期强者对决、血河追猎……这些碎片既是预兆，也是伏笔，揭示着更大的危机正在逼近。

## 主要角色

| 角色 | 身份 | 简介 |
|:-----|:-----|:-----|
| **韩林** | 太虚宗外门弟子→真传弟子 | 伪灵根废物，傲骨铮铮。实为魔帝第九世轮回身，体内封印魔帝残魂。父仇未报，三年之约在身，立誓要让所有人后悔。 |
| **柳如烟** | 太虚宗内门弟子 | 退婚未婚妻，地阶上品灵根。天之骄女，却甘愿背负骂名，以退婚保护韩林。疑似太古时期某大能转世，与韩林有三世情缘。 |
| **魔帝残魂** | 韩林体内的声音 | 梦中导师，太古魔帝的残魂碎片。沉睡时低语提示，危机时主动干预，知晓上古秘密，是韩林真正的金手指来源。 |
| **太虚子** | 太虚宗宗主，元婴期真人 | 韩林的师父，实力深不可测。疑似上古大能转世，暗中观察韩林，最终收为真传弟子。 |
| **小蝶** | 韩林的妹妹 | 看似普通孤女，实为魔帝当年留在人间的"道种"，拥有特殊体质，命运与魔帝传承紧密相连。 |
| **叶尘** | 叶家少爷，宿敌 | 表面是天才少爷，实为血河老祖的棋子，体内藏有上古魔功残片。与韩林三年之约的另一位主角。 |
| **慕容雪** | 散修联盟核心 | 亦敌亦友的神秘女子，体内蕴含魔道气息，身份成谜。与韩林有魔气共鸣现象。 |

## 核心看点

| 看点 | 说明 |
|------|------|
| **三线交织** | 表层热血逆袭，里层魔帝觉醒，源层记忆碎片预知未来，三条线层层递进，悬念迭起 |
| **双功法同修** | 逆仙录（正道功法）+ 太古魔经（魔道功法），相互促进却也暗藏冲突，修仙与魔道如何平衡？ |
| **魔道代价** | 魔气虽强，但每次使用都有代价——失控风险、神识损伤、记忆模糊，使用越多，觉醒度越高，魔化越近 |
| **三年之约** | 经典热血契约，从废物到天才的逆袭之路，三年后剑指苍穹的期待感拉满 |
| **悬疑主线** | 父亲死因成谜，天玄宗阴谋浮现，血河老祖暗中布局，柳如烟退婚真相……层层迷雾待揭开 |
| **甜虐交织** | 退婚是守护，三年是等待，她为他背负骂名，他为她踏平诸天。感情线甜虐交织，赚足眼泪 |
